# 🏥 Patna Home Diagnostics

A full-stack web application for booking home pathology and diagnostic 
test appointments online. Built for patients in Patna who want 
convenient, doorstep diagnostic services.

---

## 🛠️ Tech Stack

**Frontend:** React · TypeScript · Tailwind CSS · shadcn-ui · Vite  
**Backend:** Node.js · Express.js  
**Database:** MongoDB  
**Auth:** JWT Authentication  
**Tools:** Git · ESLint · Postman

---

## ✨ Features

- 🔐 User Registration & Login (JWT Auth)
- 🧪 Browse Diagnostic Test Catalog
- 📅 Book Home Sample Collection Appointments
- 🛠️ Admin Dashboard to manage bookings & tests
- 📱 Fully Responsive UI

---

## 🚀 Getting Started

### Clone the repository
```bash
git clone https://github.com/rimjhim-27/pathology-project.git
cd pathology-project
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Backend Setup
```bash
cd backend
npm install
# Create a .env file with your MongoDB URI and JWT secret
npm start
```

### Environment Variables
Create a `.env` file in the backend folder:
```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
PORT=5000
```
